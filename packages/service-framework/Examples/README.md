# Examples

This example shows the core Service Framework running in a plain browser environment with no additional bindings:

- `plain-web` — vanilla TypeScript, `TimerScheduler`, no framework dependencies beyond the core

For examples using React, three.js, or both together, see the Examples folders in the respective packages:

- `@realitycollective/service-framework-react` — React bindings example
- `@realitycollective/service-framework-three` — three.js render-loop example
- `@realitycollective/service-framework-client` — React + three.js combined example
